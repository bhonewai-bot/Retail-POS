Create a `Receipt` component in `src/components/pos/receipt.tsx`. This is a client component that displays order details in a receipt-style layout and supports printing.

Define and export the `ReceiptOrderData` interface that the component accepts as props:

```typescript
export interface ReceiptOrderData {
  orderNumber: string;
  subtotal: number;
  tax: number;
  total: number;
  paymentMethod: string;
  createdAt: string;
  items: {
    quantity: number;
    price: number;
    total: number;
    product: { name: string; sku: string };
  }[];
}
```

The component should:

1. Accept `open`, `onOpenChange`, and `orderData: ReceiptOrderData` as props
2. Render inside a Dialog (reuse existing `@/components/ui/dialog` — Dialog, DialogContent, DialogHeader, DialogTitle, DialogFooter)
3. Receipt layout (receipt-style, monospace font for numbers, clear sections):
   - Store header: "RETAIL POS" centered, bold, with "Thank you for your purchase!" subtitle
   - Separator
   - Order info: order number and date/time formatted as "MM/DD/YYYY HH:MM"
   - Separator
   - Items table: each row shows product name on first line, then quantity x unit_price = line_total on second line (all left-aligned for name, right-aligned for amounts). Use monospace font for amounts.
   - Separator
   - Totals section: Subtotal, Tax, bold Total (right-aligned)
   - Separator
   - Payment method line: "Payment: {method}"
   - Footer: "This is a simulated receipt" centered, small text
4. Print button in DialogFooter that calls `window.print()` — this should ONLY print the receipt content, not the whole page
5. Close button in DialogFooter

For print CSS, add a `<style>` tag inside the component (or use `@media print` within the component) that:
- Hides everything except the receipt container when printing
- Sets the receipt container to full width, white background, black text
- Removes dialog chrome (no dialog overlay, no close button, no print button)
- Uses `@media print` selector scoped to a unique class on the receipt wrapper (e.g., `.receipt-printable`)
- The print layout: `@media print { body * { visibility: hidden; } .receipt-printable, .receipt-printable * { visibility: visible; } .receipt-printable { position: absolute; left: 0; top: 0; width: 100%; background: white; } }`

Use Tailwind classes for all styling. Monospace numbers via `font-mono`. The receipt should look like a thermal receipt (narrow, centered content, clear line separations).

Helper functions for formatting (define inside the file, do not extract):
- `formatCurrency(value: number): string` — formats as `$X.XX`
- `formatReceiptDate(dateStr: string): string` — formats as `MM/DD/YYYY HH:MM`
