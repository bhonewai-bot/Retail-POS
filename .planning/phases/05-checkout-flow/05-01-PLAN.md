**POST /api/orders — Create an order**

Accept JSON body: `{ items: Array<{ productId: number; quantity: number; price: number }>, subtotal: number, tax: number, total: number, paymentMethod: string }`

Validation:
- `items` must be a non-empty array — return 400 `{ error: "Items are required" }` if empty/missing
- Each item must have `productId` (positive int), `quantity` (positive int), `price` (positive number) — return 400 `{ error: "Invalid item data" }` if any missing
- `subtotal`, `tax`, `total` must be non-negative numbers — return 400 `{ error: "Invalid totals" }` if missing/invalid
- `paymentMethod` must be a non-empty string — return 400 `{ error: "Payment method is required" }` if missing

Implementation:
1. Generate `orderNumber`: Format `ORD-YYYYMMDD-XXXX` where XXXX is a random 4-digit number. Use `Date.now()` to get the date, pad with zeros.
2. Verify all products exist and have sufficient stock:
   - Query all products: `prisma.product.findMany({ where: { id: { in: productIds } } })`
   - Check each item's quantity <= product.stock. If any insufficient, return 400 `{ error: "Insufficient stock for [product name]" }`
3. Use `prisma.$transaction` to atomically:
   a. Create the Order record with all fields
   b. Create OrderItem records using `createMany` with the order's ID
   c. Decrement stock for each product: `prisma.product.update({ where: { id }, data: { stock: { decrement: quantity } } })`
4. Return 201 with the created order object including items.

Response shape:
```json
{
  "id": 1,
  "orderNumber": "ORD-20260707-0001",
  "subtotal": 45.97,
  "tax": 3.68,
  "total": 49.65,
  "paymentMethod": "Cash",
  "status": "completed",
  "items": [
    { "id": 1, "productId": 5, "quantity": 2, "price": 12.99, "total": 25.98 }
  ],
  "createdAt": "2026-07-07T..."
}
```

**GET /api/orders — List orders**

Query parameters:
- `page` (default 1, min 1)
- `pageSize` (default 20, min 1, max 100)
- `paymentMethod` (optional string) — filter by payment method
- `startDate` (optional ISO string) — filter orders created after this date
- `endDate` (optional ISO string) — filter orders created before this date

Prisma query:
- `findMany` with `include: { items: { include: { product: { select: { id, name, sku } } } } }`
- Dynamic where clause based on query params
- Order by `createdAt: 'desc'` (newest first)
- Apply `skip`/`take` for pagination
- Use `Promise.all` for `[orders, total]` pattern

Response shape:
```json
{
  "orders": [...],
  "pagination": { "page": 1, "pageSize": 20, "totalPages": 5, "total": 100 }
}
```
